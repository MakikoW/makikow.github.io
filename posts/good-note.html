如何使用你的 Termux 装 UnblockNeteaseMusic: 一次听歌的实践 ​

前言 ​

中考临近，学习的时不免会有想听歌的时候，最近在 Github 上看到了一个仓库 Unblock Netease Music，它将网易云的音源替换成第三方的音源，以绕过云村的 VIP 机制，非常好用。

DANGER

请在虚拟环境下使用。由于本文介绍的项目的特殊性，存在时效性，本文只分享个人经验。

安装过程 ​

方法 1: 「npm」 ​

  1. 在你的终端中键入以下命令来安装前置依赖（请确保你已经安装过相关的工具，如「git, nodejs, npm, yarn」）
sh
npm install @unblockneteasemusic/server
yarn add @unblockneteasemusic/server # 用 yarn 的「this way」
  1. 使用 npx 运行
sh
npx -p @unblockneteasemusic/server unblockneteasemusic

方法 2: 直接下载并运行 ​

去 Releases 找到最新版本，然后在下方的 Assets 找到符合你系统架构的可运行文件，下载并点击运行。

最后 ​

运行后，整体体验下来还是很不错的，最近网易云的播放页面 UI/UX 也有所改动，我的体验也加倍了。网易云最近也出了「一日 VIP」的功能，当然可以试一试。

有兴趣的可以去看看 Spotify，同样很好用，UI设计偏 sharp，但音源丰富，值得一试。
